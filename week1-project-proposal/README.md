# 학기 프로젝트 제안서

## 프로젝트명
StudySprint: 학기 프로젝트 일정 및 학습 관리 도우미

## 문제 정의
학기 프로젝트는 요구사항이 분산되고 일정이 자주 밀리기 때문에, 팀원 간 진행 상황과 마감일을 한눈에 보기 어렵다. 기존 일정 관리 도구는 일반적인 할 일 관리에는 유용하지만, 학기 프로젝트 특유의 마일스톤 중심 관리와 산출물 제출 흐름을 단순하게 정리하기에는 부족하다.

## 핵심 기능 3개
1. 마일스톤 기반 일정 관리
   - 16주 일정 또는 1학기-2학기 통합 일정으로 단계별 진행 상태를 관리한다.
2. 팀 작업 분담 및 진행 추적
   - 역할, 담당자, 진행률, 제출 여부를 한 화면에서 확인한다.
3. 제출용 산출물 정리
   - 주차별 결과물, 발표 자료, README, 보고서 링크를 모아 평가용 산출물을 빠르게 확인한다.

## 기술 스택
- Frontend: React, TypeScript, Vite
- Styling: CSS Modules 또는 Tailwind CSS
- Backend: Node.js, Express
- Database: SQLite 또는 PostgreSQL
- Deployment: Vercel 또는 Render
- Collaboration: Git, GitHub, GitHub Projects
