# Auto Response Workflow Draft

## Goal
새 RFC Discussion이 생성되면 기본 안내와 템플릿 링크를 자동 응답한다.

## Trigger
- `discussion.created`

## Draft Steps
1. 새 토론 제목이 RFC 형식인지 확인한다.
2. 기본 안내 댓글을 남긴다.
3. ADR 템플릿과 위키 시작 문서를 안내한다.
