# SLA Tracking Workflow Draft

## Goal
RFC 토론의 첫 응답까지 걸린 시간을 측정해 협업 SLA를 추적한다.

## Metrics
- First response time
- Resolution time
- Open threads older than SLA threshold

## Draft Steps
1. 새 토론과 첫 댓글의 차이를 계산한다.
2. SLA를 넘긴 토론을 주간 요약에 포함한다.
3. 위키의 Troubleshooting 문서에 반복 이슈를 반영한다.
