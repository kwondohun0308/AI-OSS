# Weekly Summary Workflow Draft

## Goal
이번 주에 생성된 RFC, 결론, 보류 항목을 한 번에 요약한다.

## Draft Steps
1. 새 Discussions를 수집한다.
2. 해결된 토론과 미해결 토론을 분리한다.
3. 위키와 ADR에 반영할 항목을 목록화한다.

## Output
- 신규 RFC 수
- 결론이 난 토론 수
- 다음 주로 이월된 질문
